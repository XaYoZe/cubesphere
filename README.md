# 方块戴森计划 · CubeSphere

> 体素沙盒 × 星际工厂 —— 一个纯前端网页游戏：白手起家伐木挖矿，一路电气化、自动化，最终发射太阳帆建成环绕恒星的戴森球。

纯原生 Web 技术实现（Three.js 渲染），**零构建、零依赖服务**：所有纹理为运行时 Canvas 程序化像素绘制，所有音效由 WebAudio 实时合成，不含任何外部素材文件。

## 快速开始

```
直接双击打开 index.html 即可游玩
```

- 无需安装、无需构建、无需联网（Three.js r128 已本地化于 `lib/`）
- 推荐 Chrome / Edge 等现代浏览器
- 进度每 20 秒自动保存到浏览器 localStorage

## 游戏流程（内置 18 步任务引导）

1. **生存起步**：徒手砍树 → 木板/木棍 → 工作台 → 木镐 → 圆石 → 石镐 → 熔炉 → 烧炼铁锭
2. **电气化**：研究「电磁学」→ 风力涡轮机发电 → 采矿机自动采脉
3. **自动化**：电弧熔炉 → 传送带物流 → 制造台 → 矩阵研究站量产电磁矩阵
4. **戴森球**：解锁太阳帆 → 电磁弹射器追踪太阳发射 → **100 张太阳帆建成戴森球**

其他系统：昼夜循环与星空、全局电网（缺电全厂降速）、有界水流模拟、树苗种植、工具耐久与挖掘等级、摔落伤害、掉落物磁吸拾取、机器朝向出入料、箱子与机器联动。

## 项目结构

```
├── index.html          入口（脚本按依赖顺序加载，无模块系统）
├── css/style.css       全部 UI 样式（MC 石质面板 + 科幻发光面板）
├── lib/three.min.js    Three.js r128（本地化）
├── CHANGELOG.log       变更记录
└── js/
    ├── noise.js        种子随机 / Perlin 噪声
    ├── blocks.js       方块定义
    ├── items.js        物品 / 配方 / 科技树 / 任务数据
    ├── textures.js     程序化纹理图集与像素图标
    ├── audio.js        WebAudio 程序化音效与生成式音乐
    ├── world.js        分块体素世界 / 地形生成 / 网格化 / 水流模拟
    ├── player.js       玩家物理与掉落物
    ├── machines.js     工业机器 / 电力 / 传送带模拟
    ├── sky.js          昼夜 / 星空 / 戴森云渲染
    ├── ui.js           背包 / 合成 / 科技树 / HUD
    └── main.js         主循环 / 交互 / 存档
```
